Sistema de Importação de Pedidos - Onion SA

Descrição do Sistema

O sistema permite que os usuários carreguem uma planilha de pedidos (exemplo). Após o usuário realizar o upload dessa planilha, o sistema processará os dados e apresentará as seguintes informações na tela:

Análise Gráfica:

  • Gráfico de Vendas por Região: Apresenta uma visualização gráfica das vendas com base na distribuição regional dos pedidos.
  • Gráfico de Vendas por Produto: Exibe um gráfico representando as vendas por produto, oferecendo uma visão geral das preferências de compra.

Lista de Vendas:

  • Nome do Cliente: Identificação dos clientes associados aos pedidos.
  • Produto: Descrição dos produtos adquiridos nos respectivos pedidos.
  • Valor Final com Entrega: Total do valor do pedido, incluindo os custos de entrega, se aplicável.
  • Data de Entrega: Data prevista ou efetiva de entrega dos produtos.
